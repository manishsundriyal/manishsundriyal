# V2

Second version of my personal website built with ❤️ and Gatsby


## Getting started
Clone
```
git clone --branch v2 https://github.com/manishsundriyal/manishsundriyal.git
```
Install
```
cd manishsundriyal
npm install
```
Run
```
gatsby develop
```
